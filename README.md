## Description
---------------

This is a simple Todo App built using Python. It allows users to create, read, update and delete todo items.

## Features
------------

* Create, read, update and delete todo items
* Store data in a text file for persistence
* User-friendly interface with command line options
* Customizable using configuration file

## Installation
---------------

To install the required dependencies, run the following command:

```bash
pip install -r requirements.txt
```

## Usage
---------

### Running the App

To run the app, navigate to the repository directory and run the following command:

```bash
python main.py
```

This will start the app in interactive mode. You can create, read, update and delete todo items using the provided commands.
