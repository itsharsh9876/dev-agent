# Todo App
=====================================

A simple command-line todo app written in Python.

## Features
------------

*   Create tasks with due dates
*   View all tasks
*   Delete tasks by ID
*   Mark tasks as completed
*   Save data to a file for persistence

## Installation Steps
--------------------

1.  Clone the repository: `git clone https://github.com/your_username/todo_app.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the app: `python main.py`

## Usage Examples
-----------------

*   Create a new task: `python main.py create <task_name>`
*   View all tasks: `python main.py view`
*   Delete a task by ID: `python main.py delete <task_id>`
*   Mark a task as completed: `python main.py complete <task_id>`

## Requirements
--------------

*   Python 3.8+
*   No external dependencies required
